# username-password
Metropolis Machine wants you to build new username registration system. It must:
- Only authenticate usernames which are six characters, as the format is the first three letters of a surname and three numbers (e.g. SCO123).
- Only authenticate a password which is more than eight characters.
- They also want you to test the function several times in the program.

1. Create a new Python file. Call it user_pass.py
2. Create a function called user_pass which will take parameters for a username and password (e.g. user, pass)
3. The function must check whether the username is exactly 6 characters long.
4. The function must check whether the password is more than eight characters.
5. Create several test cases to test all outcomes (e.g. user_pass('SCO123', 'Kittens1')) Include boundary / edge cases (e.g. 8 characters for password, 5 characters & 7 characters for username).
